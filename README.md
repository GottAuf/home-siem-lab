# Home SIEM Lab with Wazuh

## Overview

This project documents the design, deployment, and operation of a containerized Security Information and Event Management (SIEM) lab using Wazuh and Docker on Fedora Linux.

The objective is to build a reproducible home SOC environment for learning and demonstrating cybersecurity operations, including:

- Log collection and analysis
- Threat detection
- Vulnerability detection
- Security Configuration Assessment (SCA)
- File Integrity Monitoring (FIM)
- Detection engineering
- Threat hunting
- Incident response

## Technology Stack

- Fedora 44
- Docker
- Docker Compose
- Wazuh Manager
- Wazuh Indexer
- Wazuh Dashboard
